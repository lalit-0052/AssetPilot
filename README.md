# AssetFlow

> Enterprise Asset & Resource Management System built with the MERN Stack.

AssetFlow is a modern ERP-style web application that helps organizations efficiently manage, allocate, track, maintain, and audit their physical assets and shared resources. The platform digitizes asset management with role-based workflows, resource booking, maintenance approvals, audit cycles, and analytics.
